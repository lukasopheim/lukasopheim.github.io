Projects

This page highlights two projects that show my experience with data science, machine learning, data cleaning, model evaluation, and applying technical work to real problems.

Housing Price Prediction

For this project, I built machine learning models to predict housing prices. I cleaned the data, created new features, handled outliers, and compared multiple models. I tested Linear Regression, Random Forest, and XGBoost to see which model performed best.

This project helped me practice the full data science process, including exploratory data analysis, feature engineering, model training, and model evaluation. It also helped me understand how different models perform when working with real-world housing data.

Deep Neural Network Surrogate for American Option Pricing

For my capstone project, my team built a deep neural network to estimate American option prices. American options are harder to price than European options because they can be exercised early. The goal of the project was to make pricing faster while keeping the results close to traditional pricing methods.

The project used synthetic option data, benchmark prices, Black-Scholes values, and financial features called Greeks. Instead of predicting the full option price directly, the model learned the early exercise premium and combined it with a Black-Scholes baseline.

This project gave me experience with deep learning, financial modeling, synthetic data generation, and model evaluation. It also showed me how machine learning can be used to speed up complex pricing problems.
